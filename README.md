# Advanced Cyber Defence Simulator

A Python-based GUI cybersecurity simulator built using Tkinter.

## Features
- Simulates cyber attacks like:
  - DDoS
  - Ransomware
  - SQL Injection
  - Zero-Day attacks
  - Phishing
  - MBR attacks
- Real-time attack logs
- Security scoring system
- Budget management
- Interactive GUI dashboard

## Technologies Used
- Python
- Tkinter
- Multithreading

## Purpose
This project was developed to simulate cyberattack and defence scenarios in an interactive environment to support learning cybersecurity concepts and threat analysis.

python cyber_defense_simulator.py
